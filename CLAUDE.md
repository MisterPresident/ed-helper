# ED - Helper

## Architecture
This is a react project using vite, typescript, tailwind

## Purpose
It should guide me as a physican through a case in an emergency department.
Two inital pathways:
1. Leitsymptom: listed in ref/symptom.txt
    - Scores
    - Red-Flags
    - Differentials to rule out
    - Url to Algorithm and treatment
2. Leitdiagnose: listed in ref/diagnosis.txt
    - Review of Symptoms
    - Red-Flags
    - Rules to discharge
    - Url to treatment

## Output
Clinical report for my SAP based medical system. Example report in ref/report.txt.
If possible a hint weather
